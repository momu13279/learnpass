# 学习通作业考试管理助手测试计划与报告

## 1. 测试目的

本文档旨在对"学习通作业考试管理助手"桌面应用程序进行全面、系统的测试，验证系统各功能模块是否满足需求规格说明书中的要求。具体测试目的包括：

1. **功能验证**：验证系统各功能模块（用户登录、学习通登录、数据同步、作业管理、考试管理、课程管理、日历视图、提醒设置、数据统计）是否按照设计要求正确实现。
2. **性能评估**：评估系统在正常使用条件下的响应速度、资源占用及稳定性。
3. **安全性验证**：验证用户密码存储、Cookie管理等关键安全机制的有效性。
4. **兼容性测试**：验证系统在不同操作系统环境下的运行表现。
5. **用户体验评估**：验证界面交互是否符合设计预期，操作流程是否流畅。

## 2. 测试范围

本次测试覆盖学习通作业考试管理助手的所有功能模块，具体范围如下：

| 序号 | 模块名称 | 测试内容 |
|:---:|:---:|:---|
| 1 | 用户登录模块 | 本地用户名密码登录、密码校验、登录状态保持 |
| 2 | 学习通登录模块 | 网页登录、Cookie获取与存储、登录状态检测与失效提示 |
| 3 | 数据同步模块 | 课程列表同步、作业列表同步、考试列表同步、同步状态反馈 |
| 4 | 作业管理模块 | 作业列表展示、状态筛选、作业详情弹窗展示 |
| 5 | 考试管理模块 | 考试列表展示、状态筛选、考试详情弹窗展示 |
| 6 | 课程管理模块 | 课程列表展示、课程信息展示 |
| 7 | 日历视图模块 | 月历展示、月份切换、今天按钮、事件按日期显示、状态颜色区分、图例说明 |
| 8 | 提醒设置模块 | 提醒开关、提醒时间设置、设置持久化 |
| 9 | 数据统计模块 | 统计卡片展示、课程统计展示 |
